# Webpack сборка

Шаблон сборки на Webpack для React.

# Команды

### DevServer

Запускает девсервер, локальная папка build не существует.
```sh
npm start
```

### Build

```sh
npm run build
```

Сборка папки build. --mode=development.

### Production build

Сборка папки build. --mode=production.

```sh
npm run build:prod
```
